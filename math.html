Problem 1:

Say whether the following is true or false and support your answer by a proof.

\((\exists m \in \mathbb{N})(\exists n \in \mathbb{N})(3m + 5n = 12)\)

The above statement is false.

Proof:  Assume the problem statement is true.  Let \(m\) and \(n\) be natural numbers and let \(m\) take its least possible value, 1.

By algebra, \[\begin{align*} 3m + 5n &= 12 \\ 3(1) + 5n &\leq 12 \\ 5n &\leq 9 \end{align*}\]

By the definition of a natural number, the only possible value for \(n\) is 1.

By algebra, \[\begin{align*} 3m + 5n &= 12 \\ 3m + 5(1) &= 12 \\ 3m &= 7 \end{align*}\]

This is a contradiction as there is no possible natural number for \(m\) that will result in 3m = 7.  There is no natural number \(m\) for which a natural number \(n\) exists where \(3m + 5n = 12\).

Problem 2:

Say whether the following is true or false and support your answer by a proof: The sum of any five consecutive integers is divisible by 5 (without remainder).

The above statement is true.

Proof: Let \(S\) be a list of five consecutive integers, and let \(n\) be an arbitrary integer.  \(S\) can be expressed as:

\[S = \{n, \, n + 1, \, n + 2, \, n + 3, \, n + 4\}\]

By algebra, the sum of \(S\) can be expressed as:

\[\begin{align*} \sum S &= \sum_{i=0}^{4} (n + i)\\ &= (n + 0) + (n + 1) + (n + 2) + (n + 3) + (n + 4) \\ &= 5n + 10 \\ &= 5(n + 2) \end{align*}\]

\(5(n + 2)\) is always divisible by 5 (without remainder) for any value of \(n\).  As a result, the sum of any five consecutive integers is divisible by 5.

Problem 3:

Say whether the following is true or false and support your answer by a proof: For any integer \(n\), the number \(n^2 + n + 1\) is odd.

The above statement is true.

Proof:  For \(n \in \mathbb{Z}\), n can either be odd or even.

If \(n\) is odd, then \(n^2\) must also be odd.  \(n^2 + n + 1\) can then be expressed as odd + odd + odd, which is odd.

If \(n\) is even, then \(n^2\) must also be even.  \(n^2 + n + 1\) can then be expressed as even + even + odd, which is also odd.

Therefore, for any integer \(n\),  \(n^2 + n + 1\) is odd.

Problem 4:

Prove that every odd natural number is of one of the forms \(4n + 1\) or \(4n + 3\), where \(n\) is an integer.

Proof:  Using division theorem, let \(n\) be an arbitrary natural number and \(b,q,r\) be integers where \(b > 0\):

\[n = bq + r \; , \; 0 \leq r < b\]

Let b = 4:

\[n = 4q + r \; , \; 0 \leq r < 4\]

\(n\) must be one of the following expressions:

\[\begin{align*} n_0 &= 4q + 0 \\ n_1 &= 4q + 1 \\ n_2 &= 4q + 2 \\ n_3 &= 4q + 3\end{align*}\]

\(n_0\) and \(n_2\) will always be even, while \(n_1\) and \(n_3\) will always be odd.  Therefore, every odd natural number can be expressed as \(4n + 1\) or \(4n + 3\), where \(n\) is an integer.

Problem 5:

Prove that for any integer \(n\), at least one of the integers \(n,\, n + 2,\, n + 4\) is divisible by 3.

Proof:  Using division theorem, let \(n\) be an arbitrary integer and \(q,r\) be integers:

\[n = 3q + r \; , \; 0 \leq r < 3\]

The possible expressions of \(n\) become:

\[\begin{align*} n_0 &= 3q + 0 \\ n_1 &= 3q + 1 \\ n_2 &= 3q + 2\end{align*}\]

If \(n = n_0\), then 3 always divides \(n_0\) and \(n\) is divisible by 3.

If \(n = n_1\), then by algebra:

\[\begin{align*} n &= 3q + 1 \\ n + 2 &= 3q + 1 + 2 \\ n + 2 &= 3q + 3 \end{align*}\]

As 3 always divides \(3q + 3\), when \(n = n_1, \, n + 2\) will be divisible by 3.

If \(n = n_2\), then by algebra:

\[\begin{align*} n &= 3q + 2 \\ n + 4 &= 3q + 2 + 4 \\ n + 4 &= 3q + 6 \end{align*}\]

As 3 always divides \(3q + 6\), when \(n = n_2, \, n + 4\) will be divisible by 3.

Thus, at least one of the integers \(n,\, n + 2,\, n + 4\) is divisible by 3 for any integer \(n\).

Problem 6:

A classic unsolved problem in number theory asks if there are infinitely many pairs of "twin primes", pairs of primes separated by 2, such as 3 and 5, 11 and 13, or 71 and 73. Prove that the only prime triple (i.e. three primes, each 2 from the next) is 3, 5, 7.

Proof:  By the definition of a prime.

Let a set of three prime numbers, each 2 from the next, be expressed as \(P = \{p,\, p + 2,\, p + 4\}\).  From problem 5 above, at least one of the integers \(p,\, p + 2,\, p + 4\) is divisible by three.   As a prime number can only be divisible by 1 and itself, the only set \(P\) that can exist is one including 3.  2 is the only prime number less than 3, but if \(p = 2, \, P = \{2, 4, 6\}\) which is not a prime triple and does not include 3. Therefore, {3, 5, 7} is the only possible prime triple.

Problem 7:

Prove that for any natural number \(n, \, 2 + 2^2 + 2^3 + \ldots + 2^n = 2^{n+1} - 2\).

Proof:  By induction.

Let the above statement be expressed as:

\[\sum_{i=1}^{n} 2^i = 2^{n+1} - 2\]

Let \(n = 1\):

\[\begin{align*}\sum_{i=1}^{1} 2^i &= 2^{1+1} - 2 \\ 2^1 &= 2^2 - 2 \\ 2 &= 2 \end{align*}\]

The identity is valid for \(n = 1\).

Assume the identity holds for n, then:

\[\begin{align*}\sum_{i=1}^{n+1} 2^i &= \sum_{i=1}^{n} 2^i + 2^{n+1} \, , \, \mathrm{separating \, the \, final \, term} \\ &= 2^{n+1} - 2 + 2^{n+1} \, , \, \mathrm{by \, induction} \\ &= 2 \cdot 2^{n+1} - 2 \\ &= 2^{n+2} - 2 \\ &= 2^{(n+1)+1} - 2\end{align*}\]

The identity is valid for \(n + 1\).  The proof is complete by induction.

Problem 8:

Prove (from the definition of a limit of a sequence) that if the sequence \(\{a_n\}_{n=1}^\infty\) tends to limit \(L\) as \(n \rightarrow \infty\), then for any fixed number \(M \gt 0\), the sequence \(\{Ma_n\}_{n=1}^{\infty}\) tends to the limit \(ML\).

Proof:  Using the definition of a limit of a sequence, we know:

\[(\forall \epsilon \gt 0)(\exists m \in \mathbb{N})(\forall n \geq m) [|a_n - L| \lt \epsilon] \]

As \(\epsilon,M \gt 0 \; \text{and} \; \epsilon\,\) is arbitrary, we know that there must be an \(m\) for \(\epsilon\) divided by \(M\) where \(n \geq m\):

\[|a_n - L| \lt \frac{\epsilon}{M} \]

By algebra, for an \(m\) where \(n \geq m\) it follows:

\[\begin{align*} M|a_n - L| &\lt M\frac{\epsilon}{M} \\ M|a_n - L| &\lt \epsilon \\ |Ma_n - ML| &\lt \epsilon \end{align*}\]

Giving the definition of a limit for the sequence \(\{Ma_n\}_{n=1}^{\infty}\) tends to the limit \(ML\):

\[(\forall \epsilon \gt 0)(\exists m \in \mathbb{N})(\forall n \geq m) [|Ma_n - ML| \lt \epsilon] \]

Proof complete.

Problem 9:

Given an infinite collection \(A_n, \, n = 1,2,\ldots \,\) of intervals of the real line, their intersection is defined to be \(\cap_{n=1}^\infty \, A_n = \{x|(\forall n)(x \in A_n)\}\,\).  Give an example of a family of intervals \(A_n, \, n = 1,2,\ldots \,\) such that \(A_{n+1} \subset A_n \,\) for all \(n\) and \(\cap_{n=1}^\infty \, A_n = \emptyset \). Prove that your example has the stated property.

Statement:  For the family of intervals, \(A_n = (0, 2^{-n}):\; A_{n+1} \subset A_n \,\) and \(\cap_{n=1}^\infty \, A_n = \emptyset \) for all \(n \in \mathbb{N}\).

Proof:  First, prove \(A_{n+1} \subset A_n \,\).  Let \(A = (a, b),\, B = (c, d)\).

From the definitions of a subset and an intersection:

\[A \subset B \equiv A \cap B = A ,\, A \neq B\]
\[\begin{align*} A \cap B &= \{x\,|\,a \lt x \lt b\} \cap \{x\,|\,c \lt x \lt d\} \\ &= \{x\,|\,\text{max}(a,\,c) \lt x \lt \text{min}(b,\,d)\} \end{align*}\]

Let \(A = (0, 2^{-(n+1)}) ,\, B = (0, 2^{n})\), for \(n \in \mathbb{N}\):

\[\begin{align*} A \cap B &= \{x\,|\,0 \lt x \lt 2^{-(n+1)}\} \cap \{x\,|\,0 \lt x \lt 2^{-n}\} \\ &= \{x\,|\,\text{max}(0,\,0) \lt x \lt \text{min}(2^{-(n+1)},\,2^{-n})\} \\ &= (0,\,2^{-(n+1)}) = A \neq B \end{align*}\]

By the definition of a subset, \(A_{n+1} \subset A_{n}\), for \(n \in \mathbb{N}\).

Now prove the family intersection is an empty set.

Given the definiton of a family set:  \(\cap_{n=1}^\infty \, A_n = \{x|(\forall n)(x \in A_n)\}\,\)

As shown above, \(A_{n+1} \cap A_{n} = A_{n+1}\) and \(A_{n+1} \subset A_{n}\); as \(n\) increases the family intersection's least upper bound is replaced by each corresponding \(2^{-n}\):

\[\begin{align*} \cap_{n=1}^1 \, A_n = (0, 2^{-1}) &= 0 \lt x \lt \frac{1}{2} \\ \cap_{n=1}^2 \, A_n = (0, 2^{-2}) &= 0 \lt x \lt \frac{1}{4} \\ \cap_{n=1}^3 \, A_n = (0, 2^{-3}) &= 0 \lt x \lt \frac{1}{8} \\ \cap_{n=1}^k \, A_n = (0, 2^{-k}) &= 0 \lt x \lt \frac{1}{2^k} \end{align*}\]

 This reduction occurs because:

\[\lim_{n\to\infty} 2^{-n} = 0\]

As \(2^{-n}\) converges to 0, the least upper bound of the family intersection's interval must be 0.  But as shown above, x must be greater than 0.  This means the family intersection must be an empty set. \[\cap_{n=1}^\infty \, A_n = \emptyset \]

Proof complete:  \(A_n = (0, 2^{-n})\) has the properties, \(A_{n+1} \subset A_n \,\) and \(\cap_{n=1}^\infty \, A_n = \emptyset \) for all \(n \in \mathbb{N}\).

Problem 10:

Give an example of a family of intervals \(A_n, \, n = 1,2, \ldots \,\), such that \(A_{n+1} \subset A_n \,\) for all \(n\) and \(\cap_{n=1}^\infty \, A_n \,\) consists of a single real number. Prove that your example has the stated property.

Statement:  For the family of intervals, \(A_n = [0, 2^{-n}]:\; A_{n+1} \subset A_n \,\) and \(\cap_{n=1}^\infty \, A_n = \{0\} \) for all \(n \in \mathbb{N}\).

Proof:  Similar to problem 9, but this family conists of closed intervals.  First, prove \(A_{n+1} \subset A_n \,\).

Reference the definitions of a subset and an intersection in problem 9.

Let \(A = [0, 2^{-(n+1)}] ,\, B = [0, 2^{n}]\), for \(n \in \mathbb{N}\):

\[\begin{align*} A \cap B &= \{x\,|\,0 \lt x \lt 2^{-(n+1)}\} \cap \{x\,|\,0 \lt x \lt 2^{-n}\} \\ &= \{x\,|\,\text{max}(0,\,0) \lt x \lt \text{min}(2^{-(n+1)},\,2^{-n})\} \\ &= [0,\,2^{-(n+1)}] = A \neq B \end{align*}\]

Bythe definition of a subset, \(A_{n+1} \subset A_{n}\), for \(n \in \mathbb{N}\).

Now prove the family intersection is an empty set.

Given the definiton of a family set:  \(\cap_{n=1}^\infty \, A_n = \{x|(\forall n)(x \in A_n)\}\,\)

As shown above, \(A_{n+1} \cap A_{n} = A_{n+1}\) and \(A_{n+1} \subset A_{n}\); as \(n\) increases the family intersection's least upper bound is replaced by each corresponding \(2^{-n}\):

\[\begin{align*} \cap_{n=1}^1 \, A_n = [0, 2^{-1}] &= 0 \leq x \leq \frac{1}{2} \\ \cap_{n=1}^2 \, A_n = [0, 2^{-2}] &= 0 \leq x \leq \frac{1}{4} \\ \cap_{n=1}^3 \, A_n = [0, 2^{-3}] &= 0 \leq x \leq \frac{1}{8} \\ \cap_{n=1}^k \, A_n = [0, 2^{-k}] &= 0 \leq x \leq \frac{1}{2^k} \end{align*}\]

 This reduction occurs because:

\[\lim_{n\to\infty} 2^{-n} = 0\]

As \(2^{-n}\) converges to 0, the least upper bound of the family intersection's interval must be 0.  Unlike problem 9, x must be greater than or equal to 0.  This means \(0 \leq x \leq 0\); the family intersection consists of a single real number, 0. \[\cap_{n=1}^\infty \, A_n = \{0\} \]

Proof complete:  \(A_n = (0, 2^{-n})\) has the properties, \(A_{n+1} \subset A_n \,\) and \(\cap_{n=1}^\infty \, A_n = \{0\} \) for all \(n \in \mathbb{N}\).
